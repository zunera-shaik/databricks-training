# 🚀 Day 3 – SQL Joins Assignment

## 📘 Overview
This assignment focuses on mastering SQL Joins using a simple online learning platform database.  
The tasks cover different types of joins and handling unmatched records between tables.

# 🧩 Database Tables

# 📚 Assignment Highlights

✅ Display students with enrolled courses
✅ Find courses without enrollments
✅ Identify students not enrolled in any course
✅ Work with LEFT, RIGHT and FULL OUTER JOIN
✅ Generate complete reports with student, course and instructor details
✅ Bonus CROSS JOIN challenge

---

# ⚡ Special Note

🚨 MySQL does not support `FULL OUTER JOIN` directly.

Alternative used:

```sql
LEFT JOIN + RIGHT JOIN + UNION
```
